Apple iPhone Webpage Clone

A front-end clone of the Apple iPhone webpage built using HTML and CSS. This project replicates the layout, design, and structure of Apple’s official website, focusing on clean UI, multiple sections, and modern web design practices.

🚀 Features
Apple-style navigation bar with menu and icons
iPhone product showcase section with video integration
Feature cards highlighting key iPhone capabilities
Product lineup section with multiple iPhone models
Grid-based comparison layout for features
“Why Apple” section with informational cards
Accessories and additional product sections
Multi-section structured layout with clean design
Detailed multi-column footer similar to Apple website

👉 All sections are structured using HTML and styled using CSS for a visually consistent layout

🛠️ Technologies Used
HTML5
CSS3
Google Fonts (SF Pro Display)
